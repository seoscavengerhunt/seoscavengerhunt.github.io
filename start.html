How to complete the SEO challenge

As a primer, SEO is a way of designing your website so that it ranks higher on Google. It's pretty important as it's a great way to obtain free traffic from search engines.

Every page will contain clues on how to get to the next step of the challenge. Some of them may require to figure out an answer to a question, in which case you'll put that answer to the end of the base URL. So if the answer to a question is "ilovebrian" then you'd navigate to "https://seoloveletter.github.io/ilovebrian".

This challenge will be much easier solved if you're on a computer. I'm not sure if you can do this while you're on your phone, but if you do, let me know.

My goodbye letter will be dotted throughout these pages, so if you want to read my goodbye letter in it's entirety, you should finish the challenge!
